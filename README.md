# Simple Console Calculator
This python code allows us to perform operations as in caculator
## Features
The calculator supports the following operations:
1.  Add: Addition of two numbers.
2.  Subtract: Subtraction of two numbers.
3.  Multiply: Multiplication of two numbers.
4.  Divide: Division (quotient and remainder) of two numbers.
5.  Square root: Square root of the number.
6.  Power: Exponentiation (first number raised to the power of the second number).
7.  Sin value: Sine of the first number (expects input in radians, requires the math module).
8.  Cos value: Cosine of the first number (expects input in radians, requires the math module).
9.  Tan value: Tangent of the first number (expects input in radians, requires the math module).
10. End: Terminates the program.
 ## Prerequisites
To run this script, you need to have Python 3 installed on your system.

Additionally, the script uses the built-in math module for trigonometric functions. It must be imported at the beginning of the script.
## Setup
Before running the script, ensure the following line is added to the very beginning of your Python file
python
import math
